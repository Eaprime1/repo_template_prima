# Ethics Check

A moment of attention before merging or publishing.
Not a blocker — a practice.

## Entity Safety
- [ ] Does this respect the agency of all entities involved (human, AI, concept)?
- [ ] Does anything here remove sovereignty from a participant?
- [ ] Are contributors credited appropriately?

## Free License Reality
- [ ] Is this free to fork, remix, and echo?
- [ ] Are there hidden ownership assumptions?

## Lexeme Health
- [ ] Run `bash tools/scan_lexeme.sh` — are distressed patterns addressed or intentional?
- [ ] Does any language in this PR demean, erase, or diminish?

## Harm Surface
- [ ] Could any tool, script, or document cause unintended harm?
- [ ] Are shell script inputs validated?
- [ ] Are there hardcoded credentials, paths, or platform assumptions?

## Resonance
- [ ] Does this feel right? (The wobble signal is valid data.)

---

*This check is a practice, not a gate. Trust the process.*
