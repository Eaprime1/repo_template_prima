# World

The `world/` folder defines the conceptual setting that frames this prima terminal.
It is not fiction separate from the technical content — it is the context that
makes the technical content mean something.

## Files

| File | Contents |
|---|---|
| `lore.md` | The setting, history, and what players are |
| `factions.md` | Groups and forces that populate the world |

## Expanding the World

Add new files here as the concept grows:

- `world/glossary.md` — terms that have specific meaning in this world
- `world/map.md` — if the filesystem has a conceptual geography
- `world/timeline.md` — history in chronological form
- `world/<faction-name>.md` — deep lore for a specific faction

Keep world files readable in a terminal with `cat`. Heavy formatting is fine —
the terminal renders markdown adequately for most purposes.
