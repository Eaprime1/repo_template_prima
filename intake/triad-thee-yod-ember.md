# The THEE / YOD / EMBER Triad

```
THEE / YOD / EMBER
Door / Mark / Warmth
Unknown / Inception / Care
```

## THEE — The Door

The first ignored article. Shadow-gathering node.
Receives the unknown without judgment.
Asks one question: **The what?**

```bash
bash tools/thee.sh "fragment"
```

## YOD — The Mark

The smallest first mark. Smallest true action.
Small enough to survive repo-entry safety checks.
Answers: **Begin here.**

```bash
bash tools/yod.sh "smallest true action"
```

## EMBER — The Warmth

Warms a fragment until it can be named.
Keeps it alive until it has a next turn.
Does not push. Does not rush.

```bash
bash tools/ember.sh "fragment to warm"
```

## The Engine

```
Dialogue becomes inception.
Inception becomes lexeme.
Lexeme becomes character.
Character becomes tool.
Tool becomes branch.
Branch becomes memory.
Memory becomes Pandora.
```

## What This Is Not

This is not a ticketing system. Not a todo list. Not a pipeline.
It is a **listening practice** — a way to let the unknown arrive
before forcing it to be useful.
